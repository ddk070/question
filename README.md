# ✨前言
有時候人生會遇到各式各樣的問題，希望能從發問中尋找到答案。<br>
不管是甚麼問題。<br>
如果你也想問題也可以在issues一起發問，尋求答案。<br>
或是留下你的想法與見解

# 🛠️使用說明
使用issues發問，或使用issues回答你的個人想法及建議提供幫助。<br>
並用標籤分類問題。
> 1. 人際關係
> 2. 自我成長
> 3. 學業職涯
> 4. 生活與現實壓力
> 5. 興趣與創作
> 6. 其他

# 🔎注意
但在發問的同時，自己也要去尋找答案。<br>
發問只是為了得到更多不同想法與建議而已。

# ✒️建立儲存庫的原因
希望大家能互相交流想法<br>
並解決大家的煩惱或問題<br>
也希望每個人留下的問題與想法<br>
能對未來可能有相同煩惱或問題的人一些幫助！！！
